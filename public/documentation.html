Welcome to the documentation page!

  • http://127.0.0.1:8080/ is the home page. Soon to be populated with more details.
  • http://127.0.0.1:8080/documentation is this page that details all the information about the API. GET method.
  • http://127.0.0.1:8080/movies contains the list of movies. GET method.
  • http://127.0.0.1:8080/movies/:title Returns movie title. GET method.
  • http://127.0.0.1:8080/genres returns a list of movie genres. GET method.
  • http://127.0.0.1:8080/genres/:name returns a specific movie genre.
  • http://127.0.0.1:8080/directors Returns a list of directors. GET method.
  • http://127.0.0.1:8080/directors/:name Returns a list of directors by name. GET method.
  • http://127.0.0.1:8080/users Returns a list of users. GET and POST method
  • http://127.0.0.1:8080/users/:username Allows a user to register, delete their own account, update their information and search for their account by name. POST, DELETE, GET, PUT Method.
  • http://127.0.0.1:8080/users/:username Allows a user to delete their account. DELETE method.
  • http://127.0.0.1:8080/users/:userId/:movies/:movieId Allows a user to add and delete their own movies.POST and DELETE method, respectfully.
